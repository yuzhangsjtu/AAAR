# AAAR：AI加速学术研究的方法论与经验

本书面向从本科生到教授的学术研究者，系统讲解如何理性、可复现、批判性地将 AI 融入科研全流程。不堆砌工具，不回避风险，关注方法论与责任边界。

📖 **在线阅读**：https://yuzhangsjtu.github.io/AAAR/

📥 **PDF 下载**：[Releases](https://github.com/yuzhangsjtu/AAAR/releases)

## 内容结构

| 部分 | 章节 | 主题 |
|------|------|------|
| 第一部分 | 第1-3章 | 理解 AI |
| 第二部分 | 第4-9章 | AI 工具与技巧 |
| 第三部分 | 第10-15章 | 案例与实践 |
| 终章 | 第16章 | AI 时代的研究者 |
| 附录 | 第17章 | 附录 |

## 本地构建

本书使用 [Quarto](https://quarto.org/) 构建。

```bash
# 预览（实时刷新）
quarto preview

# 构建 HTML
quarto render --to html

# 构建 PDF（需要 XeLaTeX）
quarto render --to pdf
```

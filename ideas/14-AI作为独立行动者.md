## E. 需要作者补充的核心要点

### E1. 你对「AI 模拟人类被试」的态度

- 完全反对 / 有限接受 / 积极拥抱?
- 你的判断依据是什么?

### E2. 一个具体的 LLM-as-agent 研究案例

- 你做的或读到的:用 LLM 跑博弈、谈判、认知实验、社会模拟、市场仿真…
- 哪个让你印象深刻?哪个让你警觉?

### E3. LLM agent 的方法论效力

- AI 行为是否能反映人类行为? — 你引用什么文献来站队?
- 复现性、生态效度、外部效度 三个维度上 AI 模拟分别得几分?

### E4. 「行动者」概念的边界

- 把 LLM 称为「research subject」「research agent」「synthetic respondent」 — 哪个名字更准确?
- 一个 LLM 能算作研究的合作者吗?

### E5. 不同学科的应用差异

- 经济学 / 社会学 / 心理学 / 政治学 中 AI agent 的接受度?
- 自然科学(化学合成 agent、材料发现 agent)与社科 agent 的本质差异?

## F. Callout 候选位置

- `callout-tip`: 区分 LLM-as-tool / LLM-as-agent / LLM-as-subject 三种角色的判定法

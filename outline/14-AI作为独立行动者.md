# AI 作为独立行动者

走到第四级角色,我们到了一个本质上不同的地方。

前三级——RA、Supervisor、Domain Expert——AI 都是「外部工具」,你在用它做研究。本级——Agent Level——AI 不再是工具,而是**研究本身的一部分**:你在研究「AI 怎么在某个场景中行动」「AI 如何参与某个社会过程」「LLM 模拟人类行为的有效性」。

这听起来像哲学游戏,但它对方法论的影响极其具体。**当 AI 从研究的工具变成研究的对象,你的论文方法论部分要写完全不同的东西、你的伦理审查要走完全不同的流程、你的同行评议会被完全不同地评判**。这一章是 part 3 最深的一级,也是当前学术界争议最激烈的地方。


## 三种 LLM 出现在研究中的角色

要理清这个层级,先要把「AI 在研究中」的三种身份分清楚。它们经常被混用,但方法论意涵完全不同:

**身份一:LLM-as-tool(LLM 作为工具)。** 研究者用 AI 做事——筛文献、清数据、写代码、改文字。前面 9 章讨论的全部都是这种身份。**论文方法论不需要把 AI 当对象写**,只需要在使用披露处说明用了什么 AI 工具(回顾 ch08 学术诚信讨论)。

**身份二:LLM-as-agent(LLM 作为行动者)。** 研究者用 LLM 来扮演研究中的一个 actor——模拟一个被试、扮演一个谈判方、运行一个虚拟社会中的居民。**论文方法论必须把 LLM 写成研究的核心要素之一**:用了什么模型、什么版本、什么 prompt、为什么相信这个模拟反映了真实人类行为。

**身份三:LLM-as-subject(LLM 作为研究对象)。** 研究者直接研究 LLM 自己——它的能力边界、它的偏见、它的涌现行为、它的价值观。**论文方法论以 AI 行为本身为现象**,不假装这反映人类——这里 AI 是研究对象,不是研究工具或代理。

::: {.callout-tip}
## 区分三种身份的判定法

如果你在论文里写了「我们用 GPT-X 做了 Y」,问自己一个问题:**「Y 这件事如果换成『让一个研究助理做 Y』,论文的核心论点变不变?」**

- **变不大**:你处于 LLM-as-tool 模式。AI 只是省时间,论点不依赖 AI 本身的特性。
- **变得很多,因为论文要论证「AI 能像人一样」**:你处于 LLM-as-agent 模式。这个模式下你必须正面回应「为什么 AI 行为可代理人类行为」这个方法论质问。
- **完全变,因为论点就是关于 AI 本身的**:你处于 LLM-as-subject 模式。这个模式相对清晰,因为你研究的是 AI 而不是别的。
:::

混淆这三种身份是 AI 时代学术不端的新形态:**用 LLM-as-tool 的轻率态度做了 LLM-as-agent 的工作,声称结果反映人类行为**。这相当于让 AI 模拟问卷然后把结果当成实际调研结果发表——形式上没造假,但科学贡献是空的。


## 「硅样本」(silicon sample):方法论争议的核心

ch11 已经提到过 silicon sample——用 LLM 模拟人类被试。本章正面回应这个争议。

**乐观派的论证**(Argyle 等 2023 [@TODO: Argyle et al 2023] 是代表):大语言模型在大量人类文本上训练,internalize 了人类的语言、态度、行为模式。给定一个 demographic profile(年龄、性别、教育、政治倾向……),让 LLM「扮演」这个 profile 的人回答问卷,结果与真实调查数据相当一致。这意味着可以用 LLM 大规模、低成本、快速地预研问卷设计、识别可能的偏差、做 power analysis。

**怀疑派的论证**:LLM 的「行为」反映的是训练数据中**关于这个 demographic 的人类描述**,而非这个 demographic 的真实决策。当被研究的问题是「常见、被广泛讨论过的」时,模拟与真实可能高度一致;但当问题是「新颖、未被广泛讨论的、文化特殊性强的」时,模拟与真实可能严重偏离。更糟的是:**你常常不知道自己面对的是哪一类问题**——可能你以为问的是后者,其实是前者(模拟有效),或反之(模拟失效)。

**当前的主流学术判断**(2026 年):

- 作为**预研工具**(在做大规模真实实验前看大致方向),硅样本可以接受
- 作为**正式数据替代**(论文的 main analysis),硅样本目前不被主流期刊接受
- 作为**测量 LLM 偏见的研究对象**(LLM-as-subject),完全合法
- **混淆使用**(声称硅样本等同真实样本)→ 学术不端

这里关键的区分是:**硅样本作为方法是否被接受,取决于你怎么定位它**。同样是「我用 GPT 模拟了 1000 个调查回答」,如果你说「这是预研,后面有真实数据」,可接受;如果你说「这是 main analysis 的样本」,不可接受;如果你说「我研究的就是 GPT 的回答模式」,完全合法。


## LLM-as-agent 在不同学科的渗透

不同学科对 LLM-as-agent 的接受度差异很大。

**经济学/博弈论**:已经有不少论文用 LLM 跑博弈实验、谈判模拟、市场仿真。主流期刊(QJE、AER)对这类研究的接受还有保留,但 working paper 圈和应用经济学期刊已经普遍接受。

**心理学/行为科学**:接受度更低。心理学的核心方法论传统是「真人被试 + 随机分组 + IRB 审批」,LLM 模拟在多数情况下不被认为是「研究人类心理」——它最多是「研究 LLM 怎么模仿人类心理描述」。

**社会学/政治学**:中等接受度,主要用于在多元社会场景下做大规模仿真——比如模拟舆论扩散、政策辩论。但实质性结论一般不被接受。

**计算社会科学**:这是 LLM-as-agent 的天然栖息地。这个领域本身就接受「用计算模拟代理真实社会过程」(早就有 agent-based modeling 传统),LLM 只是更逼真的 agent。

**自然科学**:多模态 LLM 在化学合成路径、材料发现、药物筛选上有 agent 应用——但这里的「agent」更像 ch07 编程章里的工作流自动化,而不是「替代人类被试」的 social agent。本质问题不同,争议也不同。

## 「行动者」(actor)概念的边界问题

更深一层的问题:**当我们说 LLM 是「行动者」时,我们在说什么?**

「行动者」(actor)在社会科学传统里有非常具体的内涵——它指有意图、有偏好、能在选项间做选择、能对环境作出响应的主体。一个传统的 actor 至少应该满足:**能在面对相同情境时做出不同选择**(自由意志的弱版)、**有内部状态**(信念、偏好、记忆)、**对自己的行动有某种程度的「拥有感」**(责任的雏形)。

LLM 满足这些条件吗?这是一个真正难的问题。

- LLM 在面对相同 prompt 时**确实给出不同输出**(随机采样)——但这是 actor 意义上的「自由意志」吗?还是只是温度参数?
- LLM **没有跨会话的内部状态**(每个对话开新一遍)——所以严格说它没有「记忆」「信念」「偏好」的传统含义
- LLM **不能为自己的输出承担责任**——它没有 stake、没有后果会反作用于它

按这些标准,LLM 不是真正意义上的 actor——它是「能模拟 actor 行为的统计系统」。

这个区分对方法论的实际影响:**当你声称研究「LLM agent 在博弈中的行为」时,你研究的本质上是「这个特定 LLM 在这个特定 prompt 下的统计输出模式」,不是「actor 在博弈中的行为」**。这两件事的科学贡献完全不同——前者只对该 LLM 的特性有意义,后者对一般博弈理论有意义。混淆它们是这个领域最常见的科学过度延伸。

::: {.callout-warning}
## 「我研究 LLM agent」的论文写作三个底线

如果你做这一类研究,论文里必须说清三件事,否则就有过度延伸的嫌疑:

1. **明确说明「我们研究的是 LLM 的行为模式,不是直接对应人类行为」**——除非你有独立的 validation 数据支持「这个特定情境下 LLM 行为可以代理人类行为」
2. **说明用的什么模型、什么版本、什么 prompt 模板**——否则你的研究不可复现,因为不同模型版本结果可能完全不同
3. **限定结论的边界**:「我们的发现成立于这个 LLM、这种 prompt、这类问题;不一定推广到其他 LLM 或其他问题」

这三条不是繁文缛节,是把 LLM-as-agent 研究做成科学而非流行的关键。**没有这三条,你的研究只能告诉读者「ChatGPT 在这种情况下会输出什么」——这是科学的事实,但不是科学的贡献**。
:::


## 自然科学中的 AI agent:不一样的故事

最后值得简短讨论自然科学中的 AI agent。

化学合成 agent(给一个目标分子,AI 自己规划合成路径并控制实验)、材料发现 agent(自动跑大量虚拟材料模拟,挑出有前景的候选)、药物筛选 agent(给一个靶点,AI 自动设计候选分子)——这些应用在 2024-2025 年快速成熟。

它们的方法论争议比社会科学小得多,因为:

- **结果有客观验证**:合成出来 / 没合成出来、材料性能 / 没性能、药物结合 / 不结合
- **不冒充人类被试**:agent 只是工具,只是高度自动化的工具
- **可复现性强**:同样的输入参数 + 模型版本,基本能复现(随机性来自实验本身)

所以这一类「AI agent」更接近 ch09 的「工作流自动化」延伸,而不是 ch11-13 的「替代研究角色」。它们引发的问题主要是「自动化研究的速度爆炸如何被同行评议跟上」「自动化生成的 candidate 谁来验证」——是治理问题(下一章),不是本体论问题。


## 行动者之上是治理

让我们回顾本章的几个原则:

1. **三种 LLM 出现在研究中的身份:tool / agent / subject。** 混淆这三种是 AI 时代学术不端的新形态。
2. **硅样本作为方法,取决于你怎么定位:** 预研可、正式数据替代不可、研究 LLM 本身完全合法。
3. **LLM-as-agent 在不同学科接受度不同:** 计算社会科学高,心理学低。
4. **「行动者」概念的边界:LLM 是「能模拟 actor 行为的统计系统」,不是真正的 actor。** 混淆这两件事是科学过度延伸。
5. **三个论文写作底线:声明研究 LLM 而非人类、说清模型/版本/prompt、限定结论边界。**
6. **自然科学的 AI agent 更接近工作流自动化**,主要问题是治理而非本体论。

走到这里,part 3 的前四级——RA、Supervisor、Domain Expert、Agent——都是关于个人怎么和 AI 协作。但研究不只是个人事业,它是一个学术体制。**当成千上万的研究者都在用 AI、当 AI 论文产量爆炸、当同行评议崩盘、当评价体系被 AI 重塑——这些不再是个人选择的问题,而是治理的问题**。

下一章「伦理与治理」是 part 3 的压轴。它把视野从个人选择拉到学术体制,问一个关键问题:**当个人都做对的选择(像本书前 13 章建议的),学术体制为什么仍然可能整体走偏?以及作为研究者你能做什么、应该做什么**。
